О проекте

Это сервис базы знаний с использованием ORM (Object Relation Model).

Возможности:

  • Регистрация и авторизация пользователей с Flask-Login
  • Создание, редактирование и удаление статей
  • Публикация статей для общего доступа
  • Добавление статей в избранное
  • Поиск по статьям (регистронезависимый)
  • Использование SQLAlchemy для работы с БД
  • Поддержка PostgreSQL и SQLite

Техническая информация

Используемые технологии:

  • Flask-SQLAlchemy для ORM
  • Flask-Login для управления сессиями
  • Werkzeug для хеширования паролей
  • SQLite для локальной разработки
